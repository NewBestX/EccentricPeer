Acest modul este o aplicatie JavaScript care foloseste NodeJS ca mediu de rulare.

Pentru a rula acest modul este nevoie de o baza de date MySQL.
Detaliile de conectare la aceasta se completeaza in serverConfig.json.

Se instaleaza dependentele din package.json ruland comanda: npm install

Pentru a lansa aplicatia se ruleaza comanda: node src/main.js
